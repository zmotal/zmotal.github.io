我的第一篇博客

发表于 2016-07-05   |   评论

Create a new post

搞网站十几年了,免费确实意味着不稳定和丢数据
如果只是抱着学习的念头搞博客
本机XAMPP+wordpress 应该是技术门槛和成本最低的,速度也最快
缺点就是只能自己玩,可能没人互动加上兴致过去了就放弃了

付费的话可以搞个每年50块钱左右的.com 或个性点一百左右的me啥的域名
空间我这里确实有,也确实稳定,不过价格不算很入门要168,有个用户技术交流群,可以给你学习指明方向,不过能 Google 的就不要提问了,学会学习的能力

Quick Start

Github+jekyll略显geek
你需要去看pro git这个git的入门教程和一些jekyll的使用教程
当然还要看markdown的语法教程
如果抱着学习的念头,是可以学到好多东西
而且 github支持绑定域名,这样子你可以先买域名,玩github,等懒得折腾了整个收费空间专注写内容
有个统一域名的好处就是空间随便换,域名即入口,不会让你的用户断掉联系

域名一定要国外的买,例如 godaddy 或name
千万不能买cn域名
空间国内的必须备案,国外的不能备案
